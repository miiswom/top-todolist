# Top ToDoList

A simple and lightweight to‑do list web application built with HTML, CSS and JavaScript.  
Create tasks, mark them as done, delete tasks — stay organised!

---

## Features  
- Create and add new tasks  
- Mark tasks as complete / toggle status  
- Delete tasks  
- Responsive basic UI  
- Minimal dependency overhead (vanilla JS + Webpack)  

---

## Getting Started  

### Prerequisites  
- Node.js (v14+ recommended)  
- npm (comes with Node.js)  

### Installation  
```bash
# Clone the repo
git clone https://github.com/miiswom/top-todolist.git
cd top-todolist

# Install dependencies
npm install

# Start the development server
npm run dev
npm run build
